# table-reader
This project uses a pre-built table transformer LLM to discover tables in display form and transform them to a multi-dimensional array.

## Initial Design
Use a pre-built LLM to read documents in any format, be it HTML, PDF, Word, Powerpoint or some other "display format" and transform any tables found into a 2-dimensional array.

Follow-on work will include using the same or a second pre-built LLM to create a databse from the transformed display tables.
